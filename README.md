# AuthDesktop
### Simple desktop application for demonstrating authentication using the Avalonia framework

## Get started:
- clone the repository
- build and run(.NET 8 is required)

### p.s. The server is not always available. If the 403 error is returned, try enabling VPN or configuring a proxy.
#### To set up a proxy (Win)
- add the user's environment variable "PROXY_HOST"
- if the proxy requires authorization, add the user environment variables "PROXY_LOGIN" and "PROXY_PASS"
